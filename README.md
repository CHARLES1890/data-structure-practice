# 个人编程学习笔记仓库
## 仓库简介
本仓库用于统一归档我的长期编程学习记录，包含 Python 基础、SQL 数据库、数据结构与算法、因果推断、学习日志等全套学习资料。
所有内容以 Markdown 笔记 + 可运行代码组成，记录课堂作业、刷题思路、踩坑复盘、实操案例，完整留存学习轨迹。

## 目录结构
```study-notes/
├── log/ # 日常学习总结、阶段性复盘、计划记录
├── python 学习 /  Python 基础语法、脚本案例、实操练习代码
├── 数据结构 /  数据结构课后作业、链表 / 栈 / 队列 / 树等习题笔记
├── 因果推断 /  数据分析、统计建模相关学习记录
└── sql 学习 /  SQL 基础查询、多表联查、索引、优化案例
```

## 各文件夹内容说明
### 1. python学习
- Python 基础语法、循环、函数、类与面向对象
- 实用小脚本、文件处理、数据读取实操代码
- 练习题解题思路、易错点总结

### 2. sql学习
- DDL/DML 基础语句、聚合函数、子查询、多表 JOIN
- 数据库设计、主键/索引、事务、慢SQL优化案例
- 配套练习题与建表测试SQL

### 3. 数据结构
- 数组、链表、栈、队列、二叉树、排序、查找相关课后作业
- 每节课习题完整思路拆解、代码实现、复杂度分析

### 4. 因果推断
- 统计、计量、数据分析相关理论笔记与实验记录

### 5. log
阶段性学习总结、刷题规划、知识点查漏补缺复盘

## 仓库用途
1. 个人长期学习存档，方便随时查阅回顾；
2. 完整展示从基础语言、数据库到数据结构算法的学习路线，作为求职作品集补充；
3. 持续更新刷题、实操案例，记录完整成长路径。
